Le but de l'application est de pouvoir gérer les OP (opérations) d'une association.
Pas le temps de mettre en place une BDD, donc impossible de faire des enregistrements sur un téléphone et de les récupérer sur un autre téléphone.
L'application n'est pas finie, il manque:
  -l'ajout d'avis sur une op
  -l'ajout d'une op
  -la gestion de l'op du jour (scan de carte membre pour permettre aux membres d'avoir une réduction)
  
Le package "adapter" contient toutes les classes permettant de mettre en place les éléments dans les différents ListView.
Le package "models" contient toutes les classes des "objets" qui représentent les différentes OP, les adhérents, les avis, les menus vendus...
Le reste des classes permettent le fonctionnement des différentes pages de l'application.
